# Discord Bot for Replit

A complete Discord bot with logging and moderation commands.

## Setup in Replit

1. Import this repo to Replit
2. Go to Secrets (lock icon) and add:
   - `TOKEN` = Your bot token
   - `CLIENT_ID` = Your application ID
   - `GUILD_ID` = Your test server ID
3. Click Run

## Commands

- `/ping` - Check bot latency
- `/server` - Server information
- `/user` - User information
- `/kick` - Kick a user
- `/ban` - Ban a user
- `/purge` - Delete messages
- `/setlogchannel` - Set logging channel
- `/8ball` - Magic 8-ball
- `/roll` - Roll a dice

## Features

- Message delete/edit logging
- Member join/leave logging
- 24/7 uptime with keep-alive server
